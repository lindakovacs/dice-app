# Dice app

Simple Turtle dice guessing game.

## Run in Python Sandbox (Turtle)

Use this sandbox: https://www.pythonsandbox.com/turtle

1. Open the link above.
2. Delete any starter code in the editor.
3. Open [dice-app.py](dice-app.py) from this repo.
4. Copy all code from [dice-app.py](dice-app.py) and paste it into the sandbox editor.
5. Click **Run**.

## How to play

- Use `+` and `-` to choose your guess (2 to 12).
- Click **ROLL DICE** to roll.
- If the dice sum matches your guess, you win.
- Click **PLAY AGAIN** to start another round.
